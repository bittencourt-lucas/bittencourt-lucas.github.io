Week Seven

Creating CSS3 Transitions and Animations in CSS and triggering them with JavaScript

CSS3 has brought a lot of interesting effects that can be used to create animations in a website. This week there's a brief introduction to the Transitions and the Animations using CSS3 and how to trigger these effects with JavaScript. Later on, there will be weeks that focus specifically in Transitions, Transformations, and Animations.

CSS3 may not be compatible with older browsers, or require specific prefixes to create compatibility. Internet Explorer 9 and earlier versions aren't compatible at all.
