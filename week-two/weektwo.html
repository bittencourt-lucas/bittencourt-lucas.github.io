JavaScript - Loops, Conditional Statements, Functions, Variables, Parameters, Arrays, Associative Arrays

With ES6, JavaScript now has three variable types: var, let, const

  • var works on global scope, function scope, and can be reassigned:

  • let works on function scope, block scope, and can be reassigned (also, there's a simple for loop to show how it works):

  • const works on function scope, and on block scope (also, I'm using parameters on the function and a conditional statement - and by the way, there will be data logging on the console, so make sure you press F12 to see it!):

    • JavaScript also has arrays and associative arrays, which are two different things (although very similar)

  • Arrays are sets of information ordered by number, so if I execute a for loop by number, I'll be able to retrieve the data inside of the array.
  • Associative Arrays aren't ordered, the data is retrieved by key string values instead. This type of array is discouraged, in favor of using JavaScript objects.
